  • Landing Page:

    Tela inicial contendo uma NavBar com o logo da plataforma e um botão que exibe modal para fazer login, botões para fazer login com Facebook e Google. No Corpo da página, um carousel e um breve texto!

  • Cadastro de Usuário:

    Tela para que o usuário inserir seus dados e realizar o cadastro na plataforma.

  • Perfil do Aluno:

    Tela para que o Aluno visualizar seus dados e um carousel para o mesmo ver seus projetos cadastrados.
